# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-06-16
### Added
- Initial release of QuantumCSS framework.
- Core CSS components (Button, Card, Modal, etc.).
- Utility-first CSS classes for layout and styling.
- Standardized CSS variables for Light and Dark themes.
- Quantum CLI for project scaffolding and component management.
- MCP Server for integration with AI agents.
- React and Vue component wrappers.
- Comprehensive documentation and examples.
- CI/CD workflow with GitHub Actions.
