# Components

QuantumCSS provides a comprehensive set of UI components. Each component is designed to be accessible, responsive, and easily themeable.

## Button

The button component (`.q-btn`) comes in several variants:

- `q-btn-primary`: Primary action button.
- `q-btn-secondary`: Secondary action button.
- `q-btn-outline`: Outlined button.
- `q-btn-text`: Text-only button.
- `q-btn-loading`: Shows a loading spinner.

## Card

The card component (`.q-card`) is a flexible container for content:

- `q-card-header`: Header section.
- `q-card-body`: Main content area.
- `q-card-footer`: Footer section.
- `q-card-shadow`: Adds a depth shadow.

## Modal

The modal component (`.q-modal`) provides a dialog overlay:

- `q-modal-content`: The inner container for modal content.

## Form

QuantumCSS includes styles for various form elements (`.q-form`, `.q-input`, `.q-select`, etc.) ensuring consistent look and feel across browsers.

## Utilities

In addition to components, QuantumCSS provides utility classes for:
- Spacing (`.q-m-*`, `.q-p-*`)
- Colors (`.q-text-*`, `.q-bg-*`)
- Typography (`.q-text-left`, `.q-font-bold`, etc.)
- Flexbox and Grid layouts.

Refer to [utilities.md](utilities.md) for more details.
