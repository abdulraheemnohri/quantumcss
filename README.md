QuantumCSS is designed to help developers create responsive and visually appealing web applications with ease. It provides a comprehensive set of utility classes, pre-built components, and customizable themes to streamline the development process.

## 🚀 Why QuantumCSS?

- 📦 **Modular Architecture**: Only import what you need. Keep your bundle size small.
- ⚡ **Lightning Fast**: Built for performance with zero runtime overhead.
- 🎨 **Theming Engine**: Native support for Light and Dark modes via CSS variables.
- 📱 **Mobile First**: Fully responsive components designed for all screen sizes.
- ♿ **Accessible**: High-contrast ratios and ARIA-ready component structures.
- 🛠️ **Utility-First**: Powerful spacing, color, and typography utilities for rapid styling.

## 🏁 Quick Start

### 1. Installation

Install via NPM:

```bash
npm install quantumcss
```

### 2. Import Styles

Add the main stylesheet to your project:

```css
@import 'quantumcss/dist/quantum.min.css';
```

Or via CDN:

```html
<link rel="stylesheet" href="https://cdn.quantumcss.dev/quantum@1.0.0/quantum.min.css">
```

### 3. Usage

```html
<button class="q-btn q-btn-primary">Hello QuantumCSS!</button>
```
