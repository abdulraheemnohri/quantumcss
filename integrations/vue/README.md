# QuantumCSS Vue 3 Integration

Official Vue 3 component library for QuantumCSS.

## Usage

```vue
<script setup>
import { QButton, QBadge } from '@quantumcss/vue';
</script>

<template>
  <QButton primary>Click Me</QButton>
  <QBadge variant="success">Active</QBadge>
</template>
```

*(Note: Library is currently in early bootstrap phase)*
