# Frequently Asked Questions

# General

# What is QuantumCSS?

QuantumCSS is a lightweight, modular, and modern CSS framework designed for building responsive and accessible web interfaces.

# Is QuantumCSS free to use?

Yes, QuantumCSS is licensed under the MIT License, which means it is free to use for both personal and commercial projects.

# Usage

# Can I use QuantumCSS with React/Vue/Angular?

Absolutely! QuantumCSS is a pure CSS framework and can be used with any JavaScript framework. We also provide integration examples for React in the integrations/ directory.

# How do I customize the colors?

QuantumCSS uses CSS variables for theming. You can override these variables in your own stylesheet to change the color palette. See Themes Documentation for more details.

# Troubleshooting

# Why are my utility classes not working?

Ensure that you have imported the QuantumCSS stylesheet correctly. If you are using a build tool, check your PostCSS configuration.

# How do I report a bug?

You can report bugs and request features by opening an issue on our GitHub repository (opens new window).
